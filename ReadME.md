# Movie Recommendation System with Full DevOps Pipeline

## Objective
A Movie Recommendation System deployed on AWS using a complete DevOps workflow.

## Features
- User Login
- Search Movies
- Movie Recommendations
- Movie Ratings
- Movie Details
- Amazon S3 Poster Storage
- Dockerized Application
- Jenkins CI/CD
- Terraform Infrastructure
- CloudWatch Monitoring

## Technology Stack
- Python Flask
- HTML/CSS/JavaScript
- Docker
- Jenkins
- GitHub
- Terraform
- AWS EC2
- Amazon S3
- Amazon CloudWatch
